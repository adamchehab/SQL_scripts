# SQL notes

## Databases

```sql
-- Create new databse 
CREATE DATABASE [IF NOT EXISTS] db_name1, db_name2,  ...;  

-- Delete database  
DROP DATABASE [IF EXISTS] db_name1, db_name2,  ...;  

-- Use databse  
USE dn_name;  

-- Show all databases  
SHOW DATABASES;
```

## Tables

```sql
-- Show tables
SHOW TABLES;  

-- Show tables
DROP tаble_name

-- Show table columns
DESC tаble_name;
```
